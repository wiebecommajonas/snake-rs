# snake

## How to play

```sh
$ git clone "https://github.com/wiebecommajonas/snake-rs.git"
$ cd snake-rs
$ cargo run
```

## What is this

This is a simple implementation of a snake game in Rust. Feel free to copy, modify and play around with the code.
